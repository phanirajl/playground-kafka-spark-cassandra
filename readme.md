# Playground
This is a playground project for the frameworks:
* Apache Kafka
* Apache Spark
* Apache Cassandra

# Requirements
* Docker installed
* docker-compose installed

# Start Up Environment
1. `docker-compose up`
2. start `ApplicationKafkaReader` in IntelliJ
3. start `ApplicationKafkaWriter` in IntelliJ

# Shut Down
`docker-compose stop`
