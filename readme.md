# Playground
This is a playground project for the frameworks:
* Apache Kafka
* Apache Spark
* Apache Cassandra

# Requirements
* Docker installed
* docker-compose installed

# Start Up the environment
1. Modify `docker-compose.yml` variable `KAFKA_ADVERTISED_HOST_NAME` with your docker host ip
2. `docker-compose up`
